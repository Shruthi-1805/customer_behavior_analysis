# Data Analytics Project: Customer Shopping Behavior Analysis

## Overview
This project demonstrates an end-to-end data analytics workflow, transforming raw data into meaningful insights. It includes data loading, exploratory data analysis (EDA), data cleaning, SQL querying, and visualization using Power BI. The project concludes with a report and presentation for effective communication of insights.

## Project Workflow

### 1. Data Loading
- Loaded dataset using Pandas
- Inspected structure, data types, and missing values

### 2. Exploratory Data Analysis (EDA)
- Generated statistical summaries
- Analyzed trends and distributions
- Performed correlation analysis
- Created visualizations for better understanding

### 3. Data Cleaning
- Handled missing values
- Removed duplicate records
- Fixed inconsistent formats
- Detected and treated outliers

### 4. SQL Analysis
- Imported cleaned data into database
- Executed queries including:
  - Aggregations (SUM, AVG, COUNT)
  - GROUP BY and filtering
  - Joins and subqueries
  - Business-driven analysis

### 5. Data Visualization
- Built an interactive **Power BI dashboard**
- Key features:
  - Sales trends
  - Customer segmentation
  - Product performance
  - KPI tracking

### 6. Reporting & Presentation
- Created a detailed analytical report
- Developed a presentation using Gamma AI
